## 🚀 Quick Start Guide

### Backend Startup
```bash
# Navigate to project
cd c:\Omsystems

# Apply all migrations (if not done)
.\scripts\apply-migrations.ps1

# Start backend server
npm run dev
```

**Backend runs at:** http://localhost:3000

### Frontend Startup
```bash
# Navigate to dashboard
cd c:\Omsystems\dashboard

# Start frontend server
```
